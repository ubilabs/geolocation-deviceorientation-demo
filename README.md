## Development

Have `node.js` and `npm` installed.
Install dependencies with:

```
npm install
```

Add a .env file with the content:

```
GOOGLE_MAPS_API_KEY=key
VITE_TAG_MANAGER_ID=key // internal use only
```

Start dev server with

```
npm start
```

Runs on http://localhost:5173/
